## Development Setup

```bash
# Clone your fork
git clone https://github.com/YOUR-USERNAME/WhatsApp-Slack-Bridge.git
cd WhatsApp-Slack-Bridge

# Install dependencies
cd backend && npm install && cd ..
cd frontend && npm install && cd ..

# Run backend
cd backend
node server.js

# Run frontend (in another terminal)
cd frontend
npm run dev
```
